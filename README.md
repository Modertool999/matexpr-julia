Matexpr is a Julia macro for compiling small matrix
expressions with declaration-based shape and structure metadata.

See the project website for the user documentation and full writeup:

<https://modertool999.github.io/matexpr-julia/dev/>

Useful local commands:

```bash
julia --project=. test/runtests.jl
julia --project=bench bench/benchmark.jl
julia --project=docs docs/make.jl
```
